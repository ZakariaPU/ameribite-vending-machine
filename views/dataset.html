VENDING MACHINE DATASET

Status Device ID Machine Location Product Category Transaction TransDate Type RCoil RPrice TransTotal

COLUMN DESCRIPTION

Column Description
Status Represents if the machine data is successfully processed
Device ID A unique electronic identifier (also called ePort) for vending machines. The machine is assigned a unique ePrt* device
Machine User-friendly machine name
Location Indicates the location of the vending machine
Product Products sold from the machine
Category Carbonated, Water, Non Carbonated, and Food
Transaction Unique identifier for each transaction
TransDate Date & time of transaction
Type Transaction type (Cash/Credit)
RCoil Coil # used to sell the product
RPrice Product price
TransTotal This is the total of all transactions that will appear on the Credit Card. A user can sell drinks for $3 and snacks for $1.5 bringing the total to $4.50
